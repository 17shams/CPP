# OOP244
# Introduction to Object-Oriented Programming

---

## Course Overview
Foundational C++ course covering the core principles of object-oriented programming including classes, dynamic memory, pointers, and file I/O.

## Topics Covered

- **C-Strings & I/O** — manual string handling, cin/cout
- **Dynamic Memory** — `new`/`delete`, heap vs stack
- **Classes & Objects** — constructors, destructors, encapsulation
- **Operator Overloading** — custom operators for user-defined types
- **Inheritance & Polymorphism** — virtual functions, abstract classes
- **Templates** — function and class templates
- **STL** — vectors, maps, algorithms, iterators
- **File I/O** — reading and writing text and CSV files
