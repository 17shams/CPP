## Topics Covered

- **Operator Overloading** — custom operators for user-defined types
- **Templates** — function and class templates, generic programming
- **Inheritance & Polymorphism** — virtual functions, abstract classes
- **STL** — vectors, maps, algorithms, iterators
- **Exception Handling** — try/catch, custom exceptions
- **Move Semantics** — rvalue references, move constructors
